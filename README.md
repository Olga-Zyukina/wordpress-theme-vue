# WordPress Theme
WordPress Theme + Vue 3 Calculator

![WordPress Logo](https://upload.wikimedia.org/wikipedia/commons/2/20/WordPress_logo.svg)

## Features

- Multi-page website
- Page About
- Page Product
- Page Blog 
- Calculator for calculating the cost and volume of goods
- Single post (share on social network block)
- Search page: search results with stylish design
- SEO: basic meta tags and standard structure
- Adaptive design

## Technologies used

-  CMS WordPress
-  HTML
-  CSS3
-  JavaScript
-  PHP
-  Vue
-  Yoast
